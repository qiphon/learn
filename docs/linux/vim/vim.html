# vim 基本操作

特点：

  • vi visual interface

  • 可视化接口

  • 类似与windows中的记事本

  • vim 支持多级撤销

  • 跨平台

  • 语法高亮

  • 支持图形界面


:w  # 保存

:q  # 推出

:!  # 强制保存

:ls  # 列出所有的文件

:n  # 下一个

:N  # 上一个

:15  # 跳到指定行

/xxx  #  从光标位置开始向后搜索 xxx 字符串

?xxx  # 从光标位置开始向前搜索
