- 探索型数据分析

- 预测型数据分析

数据分析流程及技术需求

- 定义问题： 业务分析、问题抽象
- 数据收集： 数据库、爬虫
- 数据清洗： Pandas/Numpy
- 数据分析： Pandas/Matplotlib
- 数据建模： 机器学习、数据挖掘
- 报告/决策： 文档输出、方案攥写
